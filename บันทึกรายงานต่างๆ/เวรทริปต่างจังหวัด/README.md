# คู่มือการติดตั้งและใช้งานระบบบันทึกเวรทริปต่างจังหวัด (Laser Engineer)

ระบบบันทึกการเดินทางออกต่างจังหวัดและจัดตารางเวรหมุนเวียนคิวพนักงานอัตโนมัติ สำหรับทีม Laser Engineer ออกแบบระบบความปลอดภัยแยกเป็น 2 หน้าเว็บอย่างชัดเจน

---

## สถาปัตยกรรมระบบ 2 หน้าเว็บ
1. **หน้าแสดงผลพนักงาน ([index.html](file:///g:/My%20Drive/Laser%20Engineer/%E0%B8%9A%E0%B8%B1%E0%B8%99%E0%B8%97%E0%B8%B6%E0%B8%81%E0%B8%A3%E0%B8%B2%E0%B8%A2%E0%B8%87%E0%B8%B2%E0%B8%99%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B9%86/%E0%B9%80%E0%B8%A7%E0%B8%A3%E0%B8%97%E0%B8%A3%E0%B8%B4%E0%B8%9B%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B8%88%E0%B8%B1%E0%B8%87%E0%B8%AB%E0%B8%A7%E0%B8%B1%E0%B8%94/index.html)):**
   - **โหมดดูอย่างเดียว (Read-Only):** พนักงานสามารถดูตารางลำดับคิว ดูสรุปสถิติกราฟวิเคราะห์ และประวัติการเดินทางได้เท่านั้น
   - ไม่มีปุ่มจัดลำดับคิวหรือฟอร์มกรอกข้อมูล เพื่อป้องกันการบันทึกหรือสลับตำแหน่งคิวโดยไม่ได้รับอนุญาต
   - มีปุ่ม **"จัดการหลังบ้าน (Admin)"** ที่มุมขวาบน เพื่อนำไปยังหน้าจัดการ
2. **หน้าผู้ดูแลระบบ ([admin.html](file:///g:/My%20Drive/Laser%20Engineer/%E0%B8%9A%E0%B8%B1%E0%B8%99%E0%B8%97%E0%B8%B6%E0%B8%81%E0%B8%A3%E0%B8%B2%E0%B8%A2%E0%B8%87%E0%B8%B2%E0%B8%99%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B9%86/%E0%B9%80%E0%B8%A7%E0%B8%A3%E0%B8%97%E0%B8%A3%E0%B8%B4%E0%B8%9B%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B8%88%E0%B8%B1%E0%B8%87%E0%B8%AB%E0%B8%A7%E0%B8%B1%E0%B8%94/admin.html)):**
   - **การล็อกสิทธิ์เข้าใช้งาน:** บังคับให้ป้อนรหัสผ่านก่อนเข้าใช้งาน หน้าจอจะเบลอและไม่แสดงส่วนควบคุมจนกว่าจะใส่รหัสถูกต้อง
   - รหัสผ่านผู้ดูแลระบบคือ: **`Admin1234`**
   - แอดมินสามารถบันทึกทริปต่างจังหวัดใหม่ สลับจัดตำแหน่งคิวพนักงาน (เลื่อนขึ้น-ลง/ข้าม) และสั่งรีเซ็ตจัดลำดับคิวเริ่มต้นใหม่ได้ทั้งหมด

## ขั้นตอนที่ 1: การตั้งค่า Google Sheets และ Google Apps Script (Backend)

1. สร้าง **Google Sheets** ใหม่ขึ้นมา 1 ไฟล์
2. ไปที่เมนูแถบเครื่องมือด้านบน เลือก **Extensions** (ส่วนขยาย) > **Apps Script**
3. ลบโค้ดเดิมทั้งหมดออกแล้วนำรหัสโปรแกรมจากไฟล์ [apps_script.js](file:///g:/My%20Drive/Laser%20Engineer/%E0%B8%9A%E0%B8%B1%E0%B8%99%E0%B8%97%E0%B8%B6%E0%B8%81%E0%B8%A3%E0%B8%B2%E0%B8%A2%E0%B8%87%E0%B8%B2%E0%B8%99%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B9%86/%E0%B9%80%E0%B8%A7%E0%B8%A3%E0%B8%97%E0%B8%A3%E0%B8%B4%E0%B8%9B%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B8%88%E0%B8%B1%E0%B8%87%E0%B8%AB%E0%B8%A7%E0%B8%B1%E0%B8%94/apps_script.js) ในโฟลเดอร์โครงการนี้ไปวาง
4. กดปุ่ม **Save (ไอคอนแผ่นดิสก์)**
5. คลิกปุ่ม **Deploy** สีน้ำเงิน > เลือก **New deployment**
6. กดที่ไอคอนรูปเฟือง เลือกประเภทเป็น **Web app**
7. ตั้งค่าค่ากำหนดดังนี้:
   - **Description:** บันทึกเวรทริปต่างจังหวัด API
   - **Execute as:** เลือก **Me (อีเมล Gmail ของคุณ)**
   - **Who has access:** เลือก **Anyone (ทุกคน)** (เพื่ออนุญาตให้หน้าเว็บของหน้าบ้านส่งคำขอเซฟข้อมูลได้)
8. กดปุ่ม **Deploy** จากนั้นให้อนุญาตสิทธิ์การเข้าถึงข้อมูลชีท (Authorize Access)
9. คัดลอกค่า **Web App URL** ที่ได้รับ (ลิงก์ลงท้ายด้วย `/exec`) เพื่อนำไปฝังในไฟล์หน้าเว็บ

## ขั้นตอนที่ 2: การเปิดใช้งานหน้าเว็บ (เชื่อมต่ออัตโนมัติ)

หน้าเว็บหลักจะอ่านค่า API URL เพื่อดึงคิวและประวัติการเดินทางโดยตรง หากต้องการอัปเดต Google Sheet หรือเปลี่ยนลิงก์เชื่อมต่อในอนาคต:
- ให้เปิดไฟล์ [index.html](file:///g:/My%20Drive/Laser%20Engineer/%E0%B8%9A%E0%B8%B1%E0%B8%99%E0%B8%97%E0%B8%B6%E0%B8%81%E0%B8%A3%E0%B8%B2%E0%B8%A2%E0%B8%87%E0%B8%B2%E0%B8%99%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B9%86/%E0%B9%80%E0%B8%A7%E0%B8%A3%E0%B8%97%E0%B8%A3%E0%B8%B4%E0%B8%9B%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B8%88%E0%B8%B1%E0%B8%87%E0%B8%AB%E0%B8%A7%E0%B8%B1%E0%B8%94/index.html) และ [admin.html](file:///g:/My%20Drive/Laser%20Engineer/%E0%B8%9A%E0%B8%B1%E0%B8%99%E0%B8%97%E0%B8%B6%E0%B8%81%E0%B8%A3%E0%B8%B2%E0%B8%A2%E0%B8%87%E0%B8%B2%E0%B8%99%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B9%86/%E0%B9%80%E0%B8%A7%E0%B8%A3%E0%B8%97%E0%B8%A3%E0%B8%B4%E0%B8%9B%E0%B8%95%E0%B9%88%E0%B8%B2%E0%B8%87%E0%B8%88%E0%B8%B1%E0%B8%87%E0%B8%AB%E0%B8%A7%E0%B8%B1%E0%B8%94/admin.html) ขึ้นมาแก้ไข
- นำลิงก์ Web App URL ใหม่ไปเปลี่ยนในตัวแปร `sheetsUrl` ในแท็ก `<script>` ของทั้งสองไฟล์ (ประมาณบรรทัดที่ 1202 ในหน้า `index.html` และบรรทัดที่ 460 ในหน้า `admin.html`)

## ขั้นตอนที่ 3: การโฮสต์และเผยแพร่ผ่าน GitHub Pages

เพื่อแชร์แอปพลิเคชันนี้ให้สมาชิกในทีมทุกคนเข้าใช้งานได้สะดวกผ่านอินเทอร์เน็ต:

1. สมัครใช้งานหรือลงชื่อเข้าใช้ที่เว็บ [GitHub](https://github.com)
2. สร้าง Repository ใหม่ ตั้งชื่อเป็น `laser-engineer-trip` และเลือกประเภทเป็น **Public**
3. อัปโหลดทั้ง 2 ไฟล์หลัก (`index.html` และ `admin.html`) ขึ้นไปยัง Repository
4. เข้าไปที่ **Settings (การตั้งค่า)** ของ Repository > เลือกเมนู **Pages**
5. เลือก Source เป็น **Deploy from a branch** เลือก Branch เป็น **main** (หรือ master) และกด **Save**
6. เมื่อรอสักครู่ ระบบจะสร้างลิงก์สำหรับเข้าใช้งานร่วมกันในแผนก (เช่น `https://username.github.io/laser-engineer-trip/`)
